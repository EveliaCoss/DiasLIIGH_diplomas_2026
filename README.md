# 🎓 Descripción del generador de diplomas – Dias Académicos 2026

**Autora**: Evelia Coss, Parti del código de Israel Aguilar.

**Fecha**: 4 de febrero, 2026

El generador **DiasLIIGH_diplomas_2026** está diseñado para crear diplomas personalizados y elegantes en reconocimiento a la participación y logros durante los **Días Académicos 2026**. Con este sistema podrás:

-   Automatizar la creación de diplomas con nombres, cargos y actividades específicas.
-   Incorporar logotipos institucionales y diseños oficiales de los Días Académicos.
-   Garantizar uniformidad y calidad gráfica en todos los documentos.
-   Exportar diplomas listos para impresión o distribución digital.

Este generador busca resaltar el esfuerzo académico y profesional de cada participante, ofreciendo un documento conmemorativo que refleja la excelencia y el prestigio del evento.

## Tipos de diplomas

El sistema **DiasLIIGH_diplomas_2026** permite la creación automatizada de diplomas oficiales en dos modalidades:

### 1. Diplomas de participación

-   Formato: PDF y PNG.
-   Campos en blanco para completar vía R + LaTeX:
    -   Nombre del participante
    -   Actividad realizada (Comité organizador, asistentes, ponentes de cartel, ponentes de charla oral, conferencia/ponencia magistral, evaluadores/jueces)

Objetivo: Reconocer la colaboración de toda la comunidad en los Días Académicos 2026.

### 2. Diplomas de premios

-   Formato: PDF y PNG.
-   Campos en blanco para completar vía R + LaTeX:
    -   Nombre del ganador (en blanco)
    -   Lugar (primer, segundo o tercer lugar)
    -   Tipo de participación (presentación oral o cartel)
    -   Categoría (Licenciatura o Posgrado)

Total de diplomas generados: 12

-   3 primeros lugares – presentación oral, Licenciatura
-   3 primeros lugares – presentación oral, Posgrado
-   3 primeros lugares – presentación de cartel, Licenciatura
-   3 primeros lugares – presentación de cartel, Posgrado
